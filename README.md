# Beginner-AI-Chatbot
[![Python 3.13.6](https://img.shields.io/badge/python-3.13.6-blue.svg)](https://www.python.org/downloads/release/python-3136/) [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
This repo shows a very basic terminal **chatbot** built using Google's Gemini API, which you can run locally. It’s small, beginner-friendly, and meant for learning.
---------------------
## Features: 
- Chat with Google Gemini-2.5-flash in the terminal.
- Quit anytime by typing exit, quit, or bye
- Beginner-friendly code (short, simple)

## Installation:
1. Clone this repo using,
```bash 
git clone https://github.com/Haffi-Ansari/Beginner-AI-Chatbot.git
cd Beginner-AI-Chatbot 
```
2. Install dependencies using,
```bash
pip install -r requirements.txt
```
### or 
Install the required dependencies listed un requirement.txt
3. Set up your API key
    Create a file named .env in the project folder and add:
    ```ini
    API_KEY=your_api_key_here
    ```
4. Run the code/chatbot,
    ### main.py

## Example:
Chatbot ready! Type 'quit' or 'exit' to leave
You: Hello Chatbot (Your Input)
Chatbot: Hi there, How can I help you? (Reply from Gemini model)
